### Prerequisite Dependencies Installation

#### Option A: Using Homebrew (Recommended)
```bash
brew install pkg-config qt@5 boost berkeley-db@5 openssl@3 create-dmg
```

#### Option B: Using MacPorts
```bash
sudo port selfupdate
sudo port install autoconf automake pkgconfig qt5 boost db53 openssl

Install the required modern dependencies via Homebrew:
```bash
brew install autoconf automake pkg-config qt@5 boost berkeley-db@5 openssl@3
```

### How to Compile on macOS (Sonoma & Higher)

Generate the dynamic system Makefile:
```bash
qmake *.pro CONFIG+=RELEASE
```

Build and automatically package into a custom distribution .dmg image:
```bash
make

./contrib/macdeploy/build-fancy-dmg.sh
```

Tested on macOS Sonoma 14
